All + All -

Linux指令大全

  • + - hwclock：显示硬件时钟
    • -w以系统时钟调整硬件时钟
    • -s以硬件时钟调整系统时钟
  • nano：使用nano文本编辑器
  • poweroff：关机
  • + - shutdown：关机或重启
    • -r reboot
    • -h halt
    • -c cancel
    • TIME:[时间]：指明动作时间
      • now：立刻
      • +m：相对时间之后
      • hh:mm具体时间
  • who：系统当前所有登录会话
  • + - screen：会话指令
    • -S [会话]创建新会话
    • -x [会话] 加入指定会话
    • -d[会话] 离线指定会话
    • -r [会话] 恢复指定会话
    • exit 在会话中退出
    • Ctrl+a，d 先按Ctrl+a再按d剥离会话
    • -ls 显示所有已打开会话
  • + - history：历史命令
    • HISTSIZE： 命令历史记录的条数
    • HISTFILE： 指定历史文件， 默认为~/.bash_history
    • HISTFILESIZE： 命令历史文件记录历史的条数
    • HISTTIMEFORMAT=“%F %T “ 显示时间
    • HISTIGNORE=“str1:str2:… “ 忽略string1,string2历史
    • HISTCONTROL：控制命令历史的记录方式
      • ignoredups 默认， 忽略重复的命令， 连续且相同为“ 重复”
      • ignorespace 忽略所有以空白开头的命令
      • ignoreboth 相当于ignoredups, ignorespace的组合
      • erasedups 删除重复命令
  • + - ls：列出文件或目录列表
    • ls /dir/文件通配符
  • + - cp：复制文件和目录
    • + - 注意事项
      • cp 多个文件 指定目录：指定目录必须存在
      • cp -r 源目录 指定目录：复制源目录所有文件到指定文件。指定目录不存在创建目录，指定目录为文件报错
      • cp 源文件 目标文件：复制文件到目标文件。会覆盖目标文件，用-i会提示丢失风险
    • -i：覆盖前提示
    • -n：不覆盖
    • -r -R：递归复制
    • -a：保留原来文件的元数据
    • -d：不复制原来文件只复制链接名
    • -v ：详情
    • -f：强制复制
  • + - rm：删除文件
    • -i：交互
    • -f：强制删除
    • -r：递归删除
  • + - mkdir：创建目录
    • -p：不报错自动创建各个目录
    • -v：显示详细信息
    • -m MODE 创建目录直接指定权限
  • + - file：查看文件类型
    • -b：不显示文件名
    • -f：列出文件中的文件名的文件烈性
    • -L：查看软链接对应的文件类型
  • + - data：显示日期和时间
    • + - +格式：指定格式输出
      • %Y 四位年
      • %y 两位年
      • %m 月
      • %d 日
      • %H 小时
      • %M 分钟
      • %S 秒
    • --date=“自定义时间”
  • + - cal：显示月历
    • -y 显示年历
  • halt：关机
  • + - reboot：重启
    • -f 强制重启
    • -p 切断电源
  • whoami：显示当前登录有效用户
  • w：系统当前所有登录会话及所做操作
  • + - echo：标准输出并换行
    • -E 默认功能
    • -n 不自动换行
    • + - -e 启动特殊功能
      • \a 发出警告声
      • \b 退格
      • \c 不换行
      • \n 换行，光标至行首
      • \r 不换行，光标至行首
      • \t 插入tab
      • \\ 插入\
      • \0数字 插入八进制数字
      • \x数字 插入十六进制数字
  • man：查看指令帮助文档
  • + - touch：刷新文件
    • -a：仅改变atime和ctime
    • -m：仅改变mtime和ctime
    • -t：指定atime和mtime的时间戳
    • -c：如果文件不存在，则不创建文件
  • mv：移动或重命名文件（用法基本与cp相同）
  • + - tree：显示目录树
    • -d：只显示目录
    • -L 数字：指定显示层数
    • -P 表达式：只显示匹配表达式的路径
  • + - rmdir：删除空目录
    • -p：递归删除父目录
    • -v：显示详情
